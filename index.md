### 3. Choose Your File
Find the file labeled `vanilla-shaders.zip` or similar. Click on it to download. This file contains all the shaders you need for Minecraft.

### 4. Extract the Files
Once the download is complete, locate the `.zip` file in your Downloads folder. Right-click on the file and select "Extract All..." from the context menu. Follow the instructions to extract the contents to a location of your choice.

### 5. Install OptiFine (Optional)
For the best experience, consider installing OptiFine. OptiFine enhances Minecraft's graphics and allows you to use shaders. 

- **Installation Steps:**
  1. Download OptiFine from [OptiFine's official page](https://optifine.net/downloads).
  2. Open the downloaded `.jar` file. You will need Java installed on your computer for this step.
  3. Click on "Install" and wait for the process to finish.

### 6. Move Shaders to the Right Folder
After extracting the shaders, you need to move them:

- Open Minecraft.
- Click on "Options" > "Video Settings" > "Shaders."
- Click on "Shaders Folder." This opens the directory where you need to place the shaders.
- Copy the extracted shader files from the extraction location into this folder.

### 7. Select Your Shader
Return to Minecraft and navigate back to the "Shaders" menu. You should see the shaders you just moved there.

- Click on your desired shader to select it.
- Click "Done" to apply the changes.

### 8. Enjoy
Start a new game or load an existing world. You will now experience enhanced graphics thanks to the shaders.

## 🔧 Troubleshooting
If you face issues while using vanilla-shaders, try the following solutions:
